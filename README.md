# sakina-legal

Privacy Policy and Terms of Service for the Sakina iOS and Android apps, served as static pages via GitHub Pages.

## Live URLs

- Privacy Policy: https://ibrahim7860.github.io/sakina-legal/privacy.html
- Terms of Service: https://ibrahim7860.github.io/sakina-legal/terms.html
- Landing: https://ibrahim7860.github.io/sakina-legal/

## Editing

Edit `privacy.html` or `terms.html` directly, bump the `Effective date` at the top of the file, commit, and push. GitHub Pages rebuilds on push (usually within a minute).

## Hosting

Deployed from the `main` branch root via GitHub Pages. No build step — these are plain HTML + one CSS file.

## Before full public launch

These policies were drafted as accurate, comprehensive templates tailored to Sakina's tech stack (Supabase, OpenAI, RevenueCat, Mixpanel, OneSignal, Apple/Google Sign In). They are suitable for TestFlight and initial App Store submission. Before scaling user acquisition, review with an attorney — especially the governing-law section and any jurisdictional specifics.
